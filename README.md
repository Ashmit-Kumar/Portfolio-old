# Ashmit Kumar Portfolio

A modern, interactive portfolio website built with React, featuring:
- Animated hero section with contact, projects, and resume download
- Tech stack dock with macOS-style hover effect
- Skills section with glassmorphism and border glow
- Experience and education timeline
- Grid-pattern background
- Responsive and accessible design
